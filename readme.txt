This calculator can be used as a cylinder volume calculator and surface area calculator. It also finds the lateral, base, and top surface areas.

[https://www.calculator.io/cylinder-volume-calculator/](https://www.calculator.io/cylinder-volume-calculator/)

== Usage ==

Add the Cylinder Volume Calculator shortcode to your page, post or sidebar:

`[ci_cylinder_volume_calculator]`

Add the following code to your template where you would like the Cylinder Volume Calculator to appear:

`<?php display_ci_cylinder_volume_calculator(); ?>`

1. Upload the Cylinder Volume Calculator /ci_cylinder_volume_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Cylinder Volume Calculator plugin through the "Plugins" menu in WordPress.
